# Jetpack Compose UI Kit

A modern, customizable Jetpack Compose UI library built from scratch to save Android developers time on complex, production-ready components.  

This is an **ongoing open-source project** — I'm actively building and adding new components weekly. Early contributions and feedback are welcome!

## Why this library?

Custom complex UI components in Jetpack Compose often take hours (or days) to get right — smooth animations, real-time updates, accessibility, and Material 3 compliance. This kit aims to provide beautiful, drop-in replacements for the ones developers dread building.

## Current Components

### Alert Dialog (5 variants)
- Simple Dialog
- Single Button Dialog
- Long Content Dialog
- Custom Icon Dialog
- Input Field Dialog

Fully customizable, supports Material 3 styling, and handles long text, icons, and user input smoothly.

### OTP Field (3 variants)
- Filled OTP View
- Outlined OTP View
- Underlined OTP View

Auto-focus, paste support, error states, and clean animations.

### Credit Card Input (Latest addition!)
- Real-time card visual preview that updates as you type (card number, expiry, holder name).
- Smooth card flip animation when focusing on CVV field.
- Built with clean Compose animations and state management for a premium feel.

All components are:
- Fully customizable (colors, shapes, typography)
- Material 3 compliant
- Accessible (TalkBack support)
- Lightweight and performant
